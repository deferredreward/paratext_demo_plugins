If the path does not resolve to a valid menu location, the PluginMenuEntry<T>.Location will be used as a fallback.

Parameters
windowTypeType of window whose menu hierarchy will be searched for the location.
insertAfterMenuHierarchyIf specified, each successive string should be the English menu and/or menu item name that specifies the "path" to the place in the menu hierarchy (of either the main window menu or a child window menu) after which the plugin's menu is to be inserted. For menu items that have ellipses, the ellipses may be omitted.

Property Documentation

◆ InsertAfterMenuHierarchy

IReadOnlyList<string> Paratext.PluginInterfaces.CustomMenuLocation.InsertAfterMenuHierarchy
get

If specified, each successive string should be the English menu and/or menu item name that specifies the "path" to the place in the menu hierarchy (of either the main window menu or a child window menu) after which the plugin's menu is to be inserted. ENHANCE: If last path item is the same as the previous item, it will be inserted at the top of that menu instead.

For menu items that have ellipses, the ellipses may be omitted.
